El Host de la página

Para los que no sepan qué es un Host, la definición según Wikipedia es:

El término host o anfitrión se usa en informática para referirse a las computadoras u otros dispositivos conectados a una red que proveen y utilizan servicios de ella. Comúnmente descrito como el lugar donde reside un sitio web. Un anfitrión de Internet tiene una dirección de Internet única (dirección IP) y un nombre de dominio único o nombre de anfitrión (host name).

El Host que estoy utilizando es gratuito y se llama Webhost000. Lo conocí en la carrera técnica de la prepa y a pesar de que sus recursos,su base de datos y la capacidad de datos es limitada, es la herramienta perfecta para aprender a diseñar y programar páginas web con lenguaje HTML puro.
