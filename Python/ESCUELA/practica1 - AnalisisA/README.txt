PROBLEMA DEL ADOQUINAMIENTO - SANTIAGO ARROYO LOZANO

El programa se desarrolló en Python3. Se compila de la forma:

python3 main.py <k> : k es un número entero

Con k podemos controlar el tamaño de la cuadricula el cual será 2^k.
El adoquin especial es seleccionado al azar de cualquier lugar en la cuadricula. 

SALIDA
Se imprime por terminal el arreglo bidimensional que contiene la solución. Además, se mostrará de forma más visual gracias a matplotlib. Esto último puede requerir que el programa se corra através de algún editor o IDE de Python como Spyder, VS Code o PyCharm, se puede comentar el uso de MAtplotlib para evitar errores de dependencias
